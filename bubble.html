Bubble Sort Introduction

Bubble Sort is a simple, yet foundational sorting technique used to arrange elements by repeatedly comparing adjacent pairs and swapping them if they are in the wrong order.

  • How it Works The algorithm passes through the list multiple times. In each pass, it compares each pair of adjacent items and swaps them if the first is larger than the second. This "bubbles" the largest element to its correct position at the end of each pass.
  • Complexity Analysis • Average/Worst Case: O(n²) — Quadratic time complexity due to nested loops.
    • Best Case: O(n) — Linear time when the array is already sorted.
